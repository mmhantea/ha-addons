# HanTech Home Assistant Add-ons

Custom add-on repository for Home Assistant with focus on Raspberry Pi 5 monitoring and management.

## Add-ons

| Add-on | Description |
|--------|-------------|
| [Argon Neo 5](argon-neo5/) | CPU temperature monitoring for Argon Neo 5 case (RPi5) |

Follow these guides in order:
