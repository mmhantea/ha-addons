# HanTech Home Assistant Add-ons

Custom add-on repository for Home Assistant.

## Add-ons

| Add-on | Description |
|--------|-------------|
| [Argon Neo 5](argon-neo5/) | Fan control and power button management for Argon Neo 5 case (RPi5) |

## Installation

1. Go to **Settings** → **Add-ons** → **Add-on Store**
2. Click the **three dots** (top right) → **Repositories**
3. Add this URL:
   ```
   https://github.com/mmhantea/ha-addons
   ```
4. Click **Add** → **Close**
5. The add-ons will appear in the store. Click to install.
